Signal and Image Processing Project


Overview

This repository contains the code and resources for our Signal and Image Processing project. The project explores image classification (specifically focusing on a balls dataset) using two different machine learning approaches: Random Forest and Convolutional Neural Networks (CNN).

Project Structure

The project is divided into distinct milestones, with implementations available in both Jupyter Notebooks and standalone Python scripts.

dataset-balls/: The directory containing the image dataset used for training and testing.

Milestone 2 (Random Forest)

milestone2.ipynb: Jupyter notebook detailing the Random Forest model implementation and analysis.

klakettanimara.py: Python script running the Random Forest pipeline.

Milestone 3 (CNN)

milestone3.ipynb: Jupyter notebook covering the CNN architecture, training, and evaluation.

cnn_trian_and_test.py: Python script for training and testing the CNN.

ballsCNN.pth: The saved PyTorch model weights containing the fully trained CNN.

testing_random.py: Script used for random testing and validation.


Visual Studio Code was used as the primary IDE for this project.
